# COSMOS — Sistema Solar Interactivo

## Descripción del sistema

Página web demostrativa con temática espacial desarrollada como parte de la práctica de Administración de la Configuración de Software. Muestra el Sistema Solar animado con todos sus planetas orbitando en tiempo real, efectos visuales de nebulosa, cometas, campo de estrellas y una interfaz estilo HUD cinematográfico. El usuario puede interactuar con cada planeta para ver su nombre y datos reales como diámetro y duración de órbita.

Desarrollado con HTML, CSS y JavaScript puro, sin dependencias externas.
